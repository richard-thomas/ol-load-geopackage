Example GeoPackage URLs to try:

  • https://richard-thomas.github.io/ol-load-geopackage/examples/dist/Natural_Earth_QGIS_layers_and_styles.gpkg
    - [EPSG 4326] countries/lakes/rivers from naturalearthdata.com.
  • http://www.geopackage.org/data/sample1_2.gpkg
    - [EPSG:4269] Counties of the US
  • https://www.geopackage.org/data/gdal_sample.gpkg
    - [EPSG:0,4326,32631] Test data: includes tables where SRS ID is missing from ol/proj. Setting option missingDataSrsAction=noProject will keep these tables but display them (incorrectly) near 0,0 ("Null Island")
  • https://www.geopackage.org/data/gdal_sample_v1.2_spi_nonlinear_webp_elevation.gpkg
    - [EPSG:0,4326,32631] Test data: includes WKB geometry types > 7 (not supported by OpenLayers). With these unsupported geometries, setting option failedTableLoadAction=discard allows loadGpkg() to keep processing other tables in the GeoPackage.
  • https://richard-thomas.github.io/DS_Canal/dist/OS Open Data (clipped) & styles.gpkg
    - [EPSG:3857] OS open data in SW England (QGIS-exported "Package Layers" data including layer_styles SLD table)
